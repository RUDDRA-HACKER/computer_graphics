## what is raster graphics?
- **Raster graphics** is a type of computer graphics that represents images as a grid of pixels, where each pixel contains information about its color and intensity. This method of representation allows for the creation of detailed and complex images, but it can also lead to issues such as pixelation when the image is scaled up. Raster graphics are commonly used in applications such as digital photography, web graphics, and video games. The most common file formats for raster graphics include JPEG, PNG, GIF, and BMP.

### How raster graphics work
Raster graphics work by storing image data in a two-dimensional array of pixels. Each pixel is assigned a specific color value, which can be represented in various color models such as RGB (Red, Green, Blue) or CMYK (Cyan, Magenta, Yellow, Key/Black). When an image is displayed on a screen or printed, the computer reads the pixel data and renders the image accordingly. The resolution of a raster image is determined by the number of pixels it contains, with higher resolutions providing more detail but also increasing file size.it scan the image from left to right and top to bottom, rendering each pixel based on its color value. This process allows for the creation of complex images with a wide range of colors and details, but it can also lead to issues such as pixelation when the image is scaled up or viewed at a low resolution.
### Advantages of raster graphics
1. **Detail and Complexity**: Raster graphics can represent complex images with a high level of detail, making them suitable for photographs and intricate designs.
2. **Wide Color Range**: Raster graphics can display a wide range of colors, allowing for vibrant and realistic images.
3. **Compatibility**: Raster graphics are widely supported across various platforms and applications, making them easy to use and share.
### Disadvantages of raster graphics    
1. **Pixelation**: When a raster image is scaled up, it can become pixelated, resulting in a loss of quality and clarity.
2. **Large File Size**: High-resolution raster images can have large file sizes, which can be difficult to store and transmit.
3. **Limited Scalability**: Raster graphics do not scale well, as they are resolution-dependent. This means that they can lose quality when resized, especially when enlarged.
### Basic terms related to raster graphics
- **Pixel**: The smallest unit of a raster image, representing a single point of color.
- **Resolution**: The number of pixels in an image, typically expressed as width x height (e.g., 1920x1080).
- **Color Depth**: The number of bits used to represent the color of each pixel, which determines the range of colors that can be displayed (e.g., 24-bit color allows for 16.7 million colors).
- **Compression**: The process of reducing the file size of a raster image, which can be done using lossless or lossy methods. Lossless compression retains all image data, while lossy compression sacrifices some quality for a smaller file size.
